# Ctypes-Demos
This project is useful to understand how to import, and how C++ libraries works in Python.

## Creating an external file

``` bash
g++ -shared -static -o <destiny_file>.so <c_file>.cpp
```

Obs: Must have MinGW.

## Demos

### add_two_numbers
Sums two int numbers

### sum_two_floats
Sum two float numbers
